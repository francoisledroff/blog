Javapolis 2005

21 December 2005

Tags: Java

Je reviens de javapolis 2005. C’était aussi fun, enrichissant et dense que l’année dernière.

  • Using JMX Technology Today and Tomorrow by Eamonn McManus

  • JBoss Portal by Julien Viet

  • Pragmatic SOA by John Crupi

  • Agile software development on large projects - Lessons learned by Johan Lybaert

  • The new Java Memory Model by Brian Goetz

  • Spring Update by Rod Johnson

  • Extreme Swing by Romain Guy

  • GlassFish: An Open Source Java Application Server for the Java EE 5 Platform by Inderjeet Singh

  • Web Services Architectures by Dennis Sosnoski

  • Agile Model Driven Development by Scott W. Ambler
